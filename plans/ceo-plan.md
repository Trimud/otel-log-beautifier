## Vision

### 10x Check
The 10x version is a trace-aware terminal. Instead of just formatting individual log lines, the terminal understands request flows. Click a trace_id, see every log line from that request highlighted across scrollback. OSC hyperlinks make trace IDs clickable links to Jaeger/Tempo. The terminal becomes a lightweight distributed tracing UI that requires zero infrastructure. For a developer debugging a distributed system, this is the difference between "nice formatting" and "I can actually follow what happened."
